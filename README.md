Customer Churn Prediction System

A full-stack machine learning project that predicts whether a customer is likely to churn using a trained Logistic Regression model, with an interactive React frontend and Flask backend.

Features
Real-time churn prediction
Interactive charts and visualizations
Fast API using Flask
Machine Learning model integration
User-friendly React UI
